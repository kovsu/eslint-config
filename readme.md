## Eslint config

> inspired by antfu's [eslint-config](https://github.com/antfu/eslint-config)

### Usage

```bash
npm i -D eslint @kovsu/eslint-config-(`basic`, `ts` or `vue`)
```

use it in your `.eslintrc.js`

```js
module.exports = {
  extends: "@kovsu/eslint-config-(`basic`, `ts` or `vue`)",
}
```
